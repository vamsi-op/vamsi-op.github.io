## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information
4. Replace placeholder images with your actual photos
5. Update social media links and contact information

## Customization

### Personal Information
- Edit the text content in `index.html`
- Update the `<title>` tag and meta descriptions
- Replace "Your Name" with your actual name throughout the files

### Colors and Styling
- Modify CSS custom properties in `styles.css`
- Update gradient colors in the `.background` and other gradient classes
- Adjust glass morphism effects by changing rgba values

### Content Sections
- Add or remove sections as needed
- Update project information and links
- Modify skill tags and categories
- Update social media links

### Images
- Replace the placeholder profile image
- Add actual project screenshots
- Update favicon and social media preview images
